# Sable Ragdolls

Sable Ragdolls is a mod that adds player and mob ragdolls powered by Sable's physics system.
This project is still early and experimental. The goal is to keep the core mod small while giving addon developers enough hooks to trigger ragdolls from their own gameplay.

Full documentation lives in the [project wiki](https://github.com/Leo-T22/sable-player-ragdoll/wiki).
